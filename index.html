Draft

Draft is a lightweight, Java-based text editor designed for simplicity and efficiency. Inspired by the functionality of Emacs, it provides a clean and intuitive interface for editing text files, with additional features like a built-in terminal for advanced users. It is ideal for writers and anyone who needs a fast and customizable text editing tool.

Tabard

Tabard is a Chrome extension that enhances tab management by offering features like automatic grouping by domain, advanced search, session saving, and theme customization. It's designed to help users, including developers and casual browsers, maintain an organized and efficient browsing experience.

Draft Screenshots

MacOS JAR ZIP

System Requirements:

  • MacOS Version: macOS 12+
  • RAM: 4 GB minimum (8 GB recommended for better performance)
  • Storage: 1 GB free disk space for installation
  • Additional Requirements:
    • MacOS DMG: No additional Java installation required (included in package)
    • JAR file users: Java Runtime Environment (JRE) 11 or later
    • ZIP file users: Java Runtime Environment (JRE) 11 or later

Installation Instructions:

  • MacOS DMG:
    1. Download the DMG file
    2. Double-click to open the disk image
    3. Drag the Draft application to your Applications folder
    4. Launch from your Applications folder
  • JAR file:
    1. Download the JAR file
    2. Launch using: java -jar Draft.jar
  • ZIP file:
    1. Download the ZIP file
    2. Extract contents to your desired location
    3. Run the included launcher
