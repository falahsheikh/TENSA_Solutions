Draft

Draft is a lightweight, Java-based text editor designed for simplicity and efficiency. Inspired by the functionality of Emacs, it provides a clean and intuitive interface for editing text files, with additional features like a built-in terminal for advanced users. It is ideal for writers and anyone who needs a fast and customizable text editing tool.

Tabard

Tabard is a Chrome extension that enhances tab management by offering features like automatic grouping by domain, advanced search, session saving, and theme customization. It's designed to help users, including developers and casual browsers, maintain an organized and efficient browsing experience.

System Requirements

  • Operating System: macOS 12 or later
  • RAM: 4 GB minimum (8 GB recommended)
  • Storage: 1 GB of free disk space
  • Java Requirements:
    • macOS DMG: Includes Java, no additional installation required
    • JAR & ZIP: Requires Java Runtime Environment (JRE) 11 or later

Installation Instructions

  • ZIP file (for macOS users):
    1. Download and extract the ZIP file
    2. Open the extracted Draft.app folder
    3. Double-click on Draft.app to launch the application
  • JAR file (for all platforms):
    1. Download the JAR file
    2. Ensure that you have Java Runtime Environment (JRE) 11 or later installed
    3. Run the JAR file using the command: java -jar DraftByFalah.jar
