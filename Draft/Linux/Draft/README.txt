Draft Application for Linux

========================


Requirements:

- Java 20 or later must be installed


Installation:

1. Extract all files to any location on your computer

2. Open terminal in the extracted folder

3. Make the launch script executable: chmod +x launch-draft.sh

4. Run the application: ./launch-draft.sh


Troubleshooting:


- If the application does not start, ensure Java is installed

- To check your Java version, open Terminal and type: java -version
